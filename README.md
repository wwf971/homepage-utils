# Homepage-Utils

A lightweight tool for interacting with and visualizing data in elasticsearch index, mongodb/sql databases, and file systems, for personal homepage/blog development.

The tool comes with a frontend for data visualization, and a backend for interacting with indices/databases/file systems.

##  Frontend
Frontend is implemented using React.js, utilizing React components from on [react-comp-mis](https://github.com/wwf971/react-comp-misc).

## Backend
Backend is implemented using Java Spring Boot.
